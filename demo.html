Game Requirements

Most computers should be able to run Momentum Shift without any issues!

Game Download

Click the download button and a zip file of the game will appear. Download and extract the zip file to your computer using a program such as Winrar. Once the game is successfully extracted, navigate to the Momentum shift folder on your computer. Open up the Build folder and look for the file "Momentum Shift.exe". Launch this executable and you will be able to play our game!
